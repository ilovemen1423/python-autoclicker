# Python Auto Clicker

A simple autoclicker made in Python.

## Features
- Adjustable CPS
- Keyboard keybinds
- Mouse button keybinds
- Side mouse button support
- Toggle on/off
- Dark UI

## Installation

Install Python libraries:

pip install pyautogui pynput

## Run the program

python autoclicker.py

## Controls

- Set CPS and press Enter
- Click "Set Keybind"
- Press a key or mouse button
- Press Enter to confirm
