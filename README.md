# Python Auto Clicker v1.0

## Features:
- Adjustable CPS
- Keyboard & Mouse keybinds (including side buttons)
- Toggle ON/OFF
- Dark UI

## Requirements:
- Python 3.10+ → https://www.python.org/downloads/
- Python libraries: pyautogui, pynput
    Install them with:
    ```
    pip install pyautogui pynput
    ```
- Windows / Mac / Linux (tested on Windows 10/11)

## How to Download & Use:
1. Click the **Download** button below to get `autoclicker.py`.
2. Install Python if you don’t have it.
3. Open a terminal / command prompt.
4. Navigate to the folder where `autoclicker.py` is downloaded.
5. Install the required libraries:
    ```
    pip install pyautogui pynput
    ```
6. Run the autoclicker:
    ```
    python autoclicker.py
    ```
7. Set your CPS, choose your keybind (keyboard or mouse), and enjoy!
